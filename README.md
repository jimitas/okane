# おかねのけいさん v1.0

**jimitas.com**が提供する小学生向けのお金の計算練習ツールです。ドラッグ&ドロップ操作で楽しく学習できます。

[![License](https://img.shields.io/badge/license-Educational-blue.svg)](https://jimitas.com)
[![Version](https://img.shields.io/badge/version-1.0-green.svg)](https://github.com/jimitas/okane-keisan)
[![Website](https://img.shields.io/badge/website-jimitas.com-orange.svg)](https://jimitas.com/105_okane/)

## 🎯 アプリの特徴

### 📚 3つの学習モード

#### セットモード
- **金額を入力してセット**: 1～99999円の範囲で自由に金額を設定
- **お金を並べる練習**: 指定した金額になるようにお金を配置
- **いくら？機能**: 並べたお金の合計金額を確認
- **リセット機能**: お金の配置をやり直し

#### ならべようモード
- **ランダム問題生成**: 自動で金額が出題される
- **お金を並べる**: 指定された金額になるようにお金を配置
- **答え合わせ**: 正解判定とフィードバック
- **コイン獲得**: 正解するとコインが貯まる

#### もんだいモード
- **ランダムお金配置**: 自動でお金が配置される
- **合計金額を計算**: 並んだお金の合計を計算
- **答えを入力**: 計算結果を入力欄に記入
- **答え合わせ**: 正解判定とフィードバック
- **コイン獲得**: 正解するとコインが貯まる

### 💰 対応しているお金（9種類）
- **紙幣**: 1万円、5千円、千円
- **硬貨**: 500円、100円、50円、10円、5円、1円
- **選択可能**: 使うお金の種類をチェックボックスで選択
- **初期設定**: 100円、50円、10円、5円、1円がデフォルトで選択済み

### 🎮 インタラクティブな学習体験
- **ドラッグ&ドロップ操作**: お金をドラッグして財布や表に配置
- **リアルタイムフィードバック**: 操作に応じた効果音
- **音響効果**: セット音、正解音、エラー音で学習をサポート
- **答え合わせ**: 完了時の正誤判定
- **コインシステム**: 正解するとコインが貯まる
- **やりなおし機能**: ページ全体をリセット

### 📱 全デバイス対応
- **レスポンシブデザイン**: スマホ・タブレット・PC対応
- **タッチ操作**: スマートフォンでのタッチ操作に対応
- **直感的UI**: 小学1年生でも使いやすいインターフェース

## 🏫 教育的効果

### 学習目標
1. **お金の種類を覚える**: 9種類のお金の見た目と価値を理解
2. **お金の数え方を習得**: 合計金額の計算方法を学習
3. **お金の並べ方を理解**: 効率的なお金の組み合わせ方を習得
4. **金銭感覚の育成**: 実生活で使える金銭感覚を養う
5. **計算力の向上**: 足し算の練習にも効果的

### 対象学年
- **主な対象**: 小学校1年生（お金の学習）
- 小学校2年生（復習・発展）
- お金の学習を始めたいすべての子どもたち

## 🚀 使用方法

### セットモード
1. **金額を入力**: 1～99999円の範囲で入力
2. **「セット」ボタン**: お金が自動で財布に配置される
3. **お金を並べる**: 財布から表にドラッグ&ドロップで移動
4. **「いくら？」ボタン**: 並べたお金の合計を確認
5. **「リセット」ボタン**: お金の配置をクリア

### ならべようモード
1. **「もんだい」ボタン**: ランダムに金額が出題される
2. **お金の種類を選択**: 使うお金にチェックを入れる
3. **お金を並べる**: 財布から表にドラッグ&ドロップ
4. **「こたえあわせ」ボタン**: 正誤を判定
5. **正解するとコイン獲得**: コインが1枚増える

### もんだいモード
1. **「もんだい」ボタン**: ランダムにお金が配置される
2. **合計金額を計算**: 並んだお金の合計を計算
3. **答えを入力**: 計算結果を入力欄に記入
4. **「こたえあわせ」ボタン**: 正誤を判定
5. **正解するとコイン獲得**: コインが1枚増える

### 便利機能
- **やりなおし**: ページ全体をリロードして最初から
- **コインをリセット**: 貯まったコインをリセット
- **お金の種類選択**: 使うお金を自由に選択可能

## 🛠️ 技術仕様

### フロントエンド
- **HTML5**: セマンティックマークアップ、構造化データ（Schema.org）
- **CSS3**: レスポンシブデザイン・メディアクエリ
- **JavaScript (ES6+)**: モジュール化・ドラッグ&ドロップAPI

### 主要機能
- **Drag and Drop API**: ネイティブドラッグ&ドロップ
- **Touch Events**: モバイル端末対応
- **LocalStorage**: コイン数の永続化保存
- **Audio API**: 効果音再生
- **SEO対策**: メタタグ、Open Graph、Twitter Card、構造化データ完備

### ブラウザ対応
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+
